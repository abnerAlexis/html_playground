Padding, Border, Margin

This element is created with h1 tag. It has following css injection;
min-width: 330px;
max-width: 700px;
padding: 10px; - Inner spacing to all sides.
border: 2px solid red; - Outer spacing all four sides of an element.
margin: 0 auto; - No spacing to top and bottom, auto spacing to left and right.
This is why this box is in the center of the screen.
